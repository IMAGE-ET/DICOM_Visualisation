Welcome to Segmentation tool for DICOM files.

This project is eventually going to be open-sourced, so patience young-padawan. The idea is to build and develop a solid code base which can be built upon by you guys. One of the current problems with current segmentation software is its GUI, so i've take away the GUI... sort of. I'm using OpenGL rather than VTK, performance is increased whilst being a dedicated graphics language. This makes it challenging to develop for, but the advantages are that the source code is built by developers and not users. ITK is the segmentation library being used, including its rescaling and histogram functions. The final goal is to create a GUI once all of the functionality has been developed. No more confusing interfaces!!
